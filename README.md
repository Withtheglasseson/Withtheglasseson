# SUA Hybrid Python Version

This repository contains a starter scaffold for the SUA Hybrid Python system. The structure mirrors the intended project layout and provides placeholders for the core components.

## Structure

- `main.py`: Entry point to wire together the brains and database.
- `database.py`: SQLite helper for storing triage decisions.
- `models.py`: Shared dataclass models.
- `utils/`: Supporting modules for APIs, market analysis, triage, and calibration.
- `config.yaml`: YAML configuration stub.
- `requirements.txt`: Python dependencies.

## Usage

```bash
pip install -r requirements.txt
python main.py
```
